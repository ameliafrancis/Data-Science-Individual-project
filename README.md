# Data-Science-Individual-project
This project contains the code to develop a RandomForest machine learning model using the Antibody and Nanobody Design Dataset (ANDD) to be able to predict high/low nanobody binding affinities

# Dependencies

python >= 3.8

pandas - dataframe manipulation

numpy - mathematical operations

seaborn - data visualisation

scipy - statistics

matplotlib - plotting

tqdm - progress monitoring

scikit-learn - machine learning

biopython - protein sequence analysis

# Usage

```
git clone https://github.com/ameliafrancis/Data-Science-Individual-project
cd Data-Science-Individual-project
python IndividualAnalysis.py
```
